Building an Off Canvas Menu

This project is called an Off Canvas Menu. It takes elements of a mobile UI to the web. The manu bar is hidden and on click of the hamburger menu bar, it becomes visible using a slide efect which also adjusts the content of the web page. The hamburger bars are animated and it toggles to an X when the canvas is open and back to a hamburger bar upon close. Click the hambuger bar to try it.
